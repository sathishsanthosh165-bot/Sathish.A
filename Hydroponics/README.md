# NFT Hydroponics System

## Overview
Designed, operated, and managed a Nutrient Film Technique (NFT) hydroponics system for leafy vegetable cultivation.

## Responsibilities
- Seed germination and transplanting
- EC and pH monitoring
- Nutrient solution preparation
- Crop growth monitoring
- Harvest management
